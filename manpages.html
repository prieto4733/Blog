Como buscar ayuda en las paginas man

Comandos que usaremos

man para abrir página las manuales de el comando

man nombre_del_comando

Y pa' que

Un tema importante en aprender a dominar el sistema de Linux y su terminal, es el aprender a leer y obtener ayuda de las página manuales del sistema llamadas las página man. En estas página los desarrolladores de los programas usados en Linux, nos dejan un manual para poder ayudarnos a aprender a usar el programa por medio de su comando. También nos ayudan a poder extender nuestro uso para mejores o mas avanzados usos. Hay una manera global de escribir estas página manuales. Hablaremos de la manera basica de cómo leer las paginas man y cómo usarlas.

Como usar páginas man

Cuando usamos las paginas man, ay manera especifica de navegar dentro de las paginas. Además, el sinopsis tiene un formato el cual nos ayuda a usar el programa. Antes de ver el sinopsis de otros programas, veremos como usar las páginas man.

Simplemente vamos a usar el comando man y el argumento de man, o en otras palabras, vamos a ver el manual del comando man

man man

La navegación

Dentro de la pagina de man, y antes de hacer cualquier cosa, vamos a apretar la tecla h. Entraremos en la pagina de ayuda. Aqui seremos guiados a como navegar y como usar las paginas.

Si vemos que para movernos de arriba hacia abajo, tenemos opciones. Las opciones estan a la izquierda y la explicasion a la derecha. Dentro de esas opciones existen unas comunes a linux que son la tecla j y la tecla k. Para mover de arriba hacia abajo esta la tecla j. de abajo hacia arriba usamos la tecla k. Estas teclas y sus movimientos son muy conocidas en Linux porque es el movimiento que aprendemos al usar el editor de texto VIM. Les recomiendo que aprendan los movimientos de VIM porque los usaran en mas de un programa.

NOTAS Si vemos, ay opciones que van juntas con el simbolo de ^ . Este simbolo se refiere a la tecla ctrl. Esto significa de que apretamos la tecla ctrl junto con la opcion respectiva.

Para poder brincar de arriba de la pagina hacia abajo con una tecla, podemos usar la tecla G. Si nesesitamos regresar hacia arriba con una tecla, podemos usar la teclas gg(apretando g dos veces)

Para poder salir de la pagina de ayuda usamos la tecla q. Igual, regresaremos a la pagina manual normal y usando la misma tecla q podemos salir de la paginas manuales y terminar el programa.

El formato del sinopsis

En la misma pagina de ayuda de el programa man, nos da el formato por el cual la sinopsis se escribe.

Letras fuertes Escribir las letras exactamente como se muestra
Letras subrayadas reemplazar con el argumento apropiado
[-abc] cualquier o todos los argumentos dentro de [ ] son opcionales
-a|-b opciones delimitados por | no pueden ser usados juntos
argumentos... los argumentos son repetibles
[expresiones]... todas las expresiones dentro de [ ] son repetibles
Notas Cuando tenemos tres puntos ... estos tres puntos se llaman elipsus y significa de que el argumento o la expresion se puede repetir o se puede dar mas de un argumento o expresion.
Notas No siempre van a ver [ ] como opcion. Esto significa de que el comando va a nesesitar un argumento o una expresion

Para poder entender estos conseptos nesesitamos un ejemplo. Vamos a ver las paginas man de varios comandos para poder poner estos conseptos a practica.

Ejemplos

Vamos a empesar con un comando simple que usamos mucho y es el comando de ls

Cuando vemos la pagina man de ls, vemos de que primero tenemos el nombre del comando y su explicacion. Despues tenemos el sinopsis y aqui vamos a poner atencion a como vamos a usar el comando de ls.

El sinopsis de ls va de la siguente manera:

ls [OPCION]... [ARCHIVO]...
NOTAS archivo o file puede ser el nombre de un archivo o de un directorio.

Vemos de que las opciones y el argumento estan dentro de [ ] esto significa de que es opcional usar una opcion o un argumento, entonces podemos usar el comando ls a solas

ls

Si usamos el comando de ls a solas, no va a dar la lista de archivos y directorios del el directorio en el cual residimos.

Vemos tambien de que la opciones y los argumentos estan subrayados. Significa de que las opciones y los argumentos nesesitan que reemplasarse con una opcion o argumento apropiado. En la pagina man de ls, devajo de DESCRIPCION encontramos las opciiones. Vemos de que ay maneras de escribir estas opciones, forma corta o larga. La primera opcion de ls es -a que es la manera corta de escribirlo y la manera larga de escribirlo es --all. Entonces
